Για την εργασία έχω κάνει τα πάντα μέσα στο αρχείο
Ergasia.c αλλά για τη βάση δεδομένων, επειδή ήθελα
να είναι λίγο καλύτερη, και έχω μεγάλη εξοικείωση
με Python, έγραψα ένα πρόγραμμα σε Python το οποίο
έγραψε το πρόγραμμα makedb.c που φτιάχνει μια
βάση δεδομένων 1500 ατόμων, με τυχαία ονόματα(με
χρήση λεξικού/hashmap) και διωνυμική κατανομή παιδιών
και εισοδήματος/χρέους, ώστε να υπάρχει μια μεγαλύτερη
συγκέντρωση γύρω από πιο φυσικές τιμές.

Επειδή το λειτουργικό μου είναι Linux σκέφτηκα,
σε περίπτωση κάποιας ασυμβατότητας, να συμπεριλάβω
και μερικές εικόνες για το πως τρέχει το πρόγραμμα
και πως φαίνονται τα αρχεία.

Ελπίζω να αγγίζει, έστω και λίγο, την τελειότητα!
